Lenguaje de Marcado de Hipertetxto (HTML)

Bienvenidos! Hoy estaremos platicando algunoos conceptos y elementos más importantes vistos hasta el momento de pw1, además veremos el uso de etiquetas semánticas en la estructuración de un sitio web.

Conceptos principales de HTML

HTML

Lenguaje de marcado de hipertexto, que se utiliza para hacer o estructurar una página web.

Elemento HTML

Es la unidad básica de estructura contenido de una página web, esta compuesta de:

  1. Etiqueta de aperura
  2. Pares artriuto-valor
  3. Contenido
  4. Etiqueta de cierre

Etiqueta <head>

Es una etiqueta que contiene metadatos, es la forma en que una página web se comunica con el navegador, funciona para el SEO, se puede enlazar archivos CSS o Scripts.

Etiqueta <body>

Contiene todo el contenido HTML.

Contenedores Semánticos>

<header> Etiqueta utilizada para el contenido introductorio de una página web puede contener el logo, el menú de navegación principal, el nombre de la empresa, etc.
<nav> Etiqueta utilizada para el menú de navegación principal suele contener enlaces a otras páginas relacionada o navegar por la página principal, no debe de haber más de una en la página.
<main> Etiqueta única de una página web cuyo propósito es resaltar nuestro objetivo o mensaje principal del sitio web dentro se colocará el contenido principal de nuestra página.
<section> Es una etiqueta que poco a poco se intenta que sustituya a <div> su uso es para dividir la página en secciones y agrupar contenido relacionado.
<article> Es una etiqueta que se utiliza para agrupar contenido autónomo, que si separa de otro contenido sigue teniendo coherencia normalmente se compone de un encabezado y contenido.
<footer> Es una etiqueta que normalmente se encuentra hasta el final de una página y funciona como cierre del sitio web a menudo contiene datos de contacto, derechos de autor u otra información complementaria.

Etiquetas principales en HTML

Etiqueta <a<

Es un elemento que nos va a permitir navegar a;

  • Página externas
  • Páginas internas
  • Secciones dentro de nuestra misma página (enlaces ancla)

Algunos valores específicos para el atributo href son:

href="https://wa.me/55977847278"
Abre un chat del número especificado en Whatspp
href="mailto:example@.com"
Abre una aplicación de correo, con el destinatario especificado

Etiqueta <form<

Es el medio con el que los usuarios pueden comunicarse con nuestra página web a través de este elemento, podemos recopilar información para posteriormente procesarla.
